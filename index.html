Deep Dive In Python

Build Pacman with Pygame

Introduction
Learn how to program in Python by creating a Pacman game using Pygame Zero.
Installation
Install the development environment on your system.
Basics of Python
Developing a pacman game application is a huge task, which becomes easier by breaking it up into smaller pieces. Similarly, learning python can be a huge task and requires learning many features, so we will start with the basics. As we learn python, we will also look at developing a small piece of pacman.
Program Structure
In the last section, we used the basics of python to add a map for pacman. As we learn some of python's program structure features, we will use them to add more components like a ghost and pacman to the map.
Compound Data Types
In the last we started adding to the map in pacman using some features of python's program structure. As we learn compound data types, we will use it to increase number of ghosts and dots in the map's maze.
Classes
Now we have a simple maze with a few ghosts and a pacman, but they do not move around yet, so we will learn how to build classes to define their behavioral actions and data.
Finish Game Development
We have a basic game, but there is still more that can be added. We will use the features we learned from classes, compound data types, program structures and the basics to finish developing our pacman game.
